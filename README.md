# OptionsPricing

A Python library for pricing European and American options using four different numerical methods — **Black-Scholes**, **Binomial Tree**, **Monte Carlo simulation**, and **Finite Difference** — with tools to fetch real market data, estimate historical volatility, simulate stock price paths, and benchmark the methods against each other.

## Features

- **Black-Scholes** closed-form pricing for European options, including all five Greeks (Delta, Gamma, Vega, Theta, Rho)
- **Binomial Tree** (CRR-style) pricing supporting both European and American exercise
- **Monte Carlo** simulation pricing with 95% confidence intervals
- **Finite Difference** (explicit scheme) PDE solver for European call pricing
- **Market data** fetching via `yfinance`, with historical volatility estimation
- **Stock path simulation** under geometric Brownian motion
- Convergence, error, and runtime benchmarks comparing all methods, with plots saved to `figures/`
- Full test suite validating pricing accuracy against known closed-form values

## Installation

```bash
git clone https://github.com/Dris67/OptionsPricing.git
cd OptionsPricing

# Create and activate a virtual environment
python -m venv venv

source .venv/bin/activate      # macOS/Linux
venv\Scripts\activate         # Windows

# Install dependencies
pip install -r requirements.txt
```

Key dependencies: `numpy`, `scipy`, `pandas`, `matplotlib`, `yfinance`, `pytest`.

Remember to activate the virtual environment (`source venv/bin/activate` or `venv\Scripts\activate`) each time you work on the project, and deactivate it with `deactivate` when you're done.

## Usage

### Black-Scholes (pricing + Greeks)

```python
from src.black_scholes import BlackScholes

option = BlackScholes(S=100, K=100, T=1, r=0.05, sigma=0.2)

option.call_price()   # 10.4506
option.put_price()    # 5.5735

option.delta("call")  # 0.6368
option.gamma()         # 0.0188
option.vega()          # 0.3752 (per 1% change in volatility)
option.theta("call")  # -6.4140 (per year)
option.rho("call")    # 53.2325
```

### Binomial Tree (European or American)

```python
from src.binomial import BinomialTree

tree = BinomialTree(S=100, K=100, T=1, r=0.05, sigma=0.2, steps=500)

tree.price("call")                       # European by default
tree.price("put", american=True)         # American exercise
```

### Monte Carlo simulation

```python
from src.monte_carlo import MonteCarlo

mc = MonteCarlo(S=100, K=100, T=1, r=0.05, sigma=0.2, simulations=100000)

mc.price("call")
mean, lower, upper = mc.confidence_interval("call")  # 95% CI
```

### Finite Difference (explicit scheme)

```python
from src.finite_difference import FiniteDifference

fd = FiniteDifference(S0=100, K=100, T=1, r=0.05, sigma=0.2, M=100, N=5000)
fd.explicit_call()
```

`M` is the number of stock-price grid points and `N` the number of time steps. The explicit scheme is only conditionally stable — too few time steps for a given `M` raises a `ValueError`.

### Real market data

```python
from src.market_data import MarketData
from src.volatility import HistoricalVolatility
from src.black_scholes import BlackScholes

market = MarketData("AAPL", "2025-01-01", "2026-01-01")
prices = market.get_prices()

sigma = HistoricalVolatility(prices).annualized()
S0 = float(prices.iloc[-1])

option = BlackScholes(S0, S0, 30/365, 0.05, sigma)  # ATM, 30 days to expiry
option.call_price()
```

### Simulating future stock paths

```python
from src.mc_simulator import StockSimulator

sim = StockSimulator(S0=100, r=0.05, sigma=0.2, T=1, steps=252, paths=100)
paths = sim.simulate()  # shape: (paths, steps + 1)
```

## Examples

The `examples/` directory contains runnable scripts demonstrating each model, including comparing all methods (`compare_models.py`), pricing against live market data (`real_market_example.py`), and plotting simulated future price paths (`simulate_stock_example.py`).

```bash
python -m examples.compare_models
python -m examples.real_market_example
```

## Benchmarks

The `benchmarks/` directory measures runtime, convergence, and accuracy of each method — for example, how the Binomial Tree and Monte Carlo prices converge to the Black-Scholes price as steps/simulations increase, and how each method's runtime compares. Plots are saved to `figures/`.

```bash
python -m benchmarks.benchmark
python -m benchmarks.binomial_convergence
python -m benchmarks.mc_convergence
```

## Testing

Run the full test suite with `pytest`:

```bash
python -m pytest
```

The tests validate pricing accuracy against known closed-form values, check the Greeks, confirm American options are worth at least as much as their European counterparts, verify finite-difference grid convergence and stability checks, and validate input error handling.
